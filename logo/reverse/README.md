# Reverse-engineered vectors (v0)

**Status:** provisional — good enough to complete the brand pack; refine later.

## Method
- Bitmap logos: **potrace** contour extraction and/or geometric rebuild from measurements
- Wordmarks: icon/mark + **Liberation Sans Bold** outlines (placeholder type until custom type locked)

## Files
See `logo/current/*` for shipping candidates generated from reverse pass.
Also mirrored under `logo/reverse/`.

## Do not claim
These are **not** original designer source files. Accuracy varies by complexity (Cubeage cube hardest).
