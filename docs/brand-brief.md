# Sylphx brand brief (constructed + product-aligned)

> Status: **Working SSOT**. Merges live sylphx.com positioning with existing `guidelines/*`.  
> Logos: use **existing** assets only; fill SVG gaps later without inventing a new mark.

## 1. Who we are

**Sylphx** is an AI-native developer platform company: ship apps with **one SDK** covering backend services, data isolation, AI gateway, and deploy — plus a public OSS footprint (MCP tools, TS libraries).

## 2. Audience

| Priority | Who | Feel |
|----------|-----|------|
| P0 | Indie / startup builders, AI-product engineers | Fast, modern, less vendor glue |
| P1 | OSS / MCP users | Competent, open, high-signal |
| P2 | Teams needing isolation + hibernate economics | Serious infrastructure, not a toy |

## 3. Attributes

| Attribute | Means |
|-----------|--------|
| **Unified** | One platform / one SDK / one dashboard |
| **AI-native** | Models & agents first-class, not bolt-on |
| **Precise** | Type-safe, isolation, engineering seriousness |

## 4. Personality

| Do | Don't |
|----|--------|
| Clear, technical, confident | Enterprise beige without substance |
| Developer-first wit (restrained) | Hype crypto / fake “AGI company” energy |
| Dark + blue product UI language | Random rainbow rebrands |
