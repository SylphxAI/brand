# What belongs in this brand repository

Fleet standard: [EpiowAI/brand REPO-ARCHITECTURE](https://github.com/EpiowAI/brand/blob/main/docs/REPO-ARCHITECTURE.md).

**About us / company narrative:** yes → `docs/copy/about.md`.  
**Product docs & API reference:** product/docs site, not this repo.  
**OSS README badges:** may cite brand colors/voice; feature lists stay in each repo.
